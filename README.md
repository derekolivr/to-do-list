A minimal, unobtrusive new-tab experience designed to stay clear and supportive for users who struggle to focus.

Simple Chrome new-tab extension that replaces the default new tab page with a personal to-do list featuring multiple lists, backgrounds, and theme customization.

### Current Functionality

- Maintain multiple task lists
- Mark tasks complete and restore them from the Finished list
- Set priorities, due dates, and edit tasks inline
- Persistent storage via Chrome sync when available, with localStorage fallback
- Change Background carousel with Unsplash images, custom uploads, and solid color themes
- Quick search filtering and keyboard-friendly task entry
